# 🏋️ نظام إدارة صالة حسام جم | Hossam Gym Management System

نظام إدارة شامل للصالات الرياضية مطور بتقنيات حديثة لإدارة المشتركين والبرامج التدريبية والأنظمة الغذائية والمبيعات.

### 👥 إدارة المشتركين

- ✅ إضافة وتعديل وحذف المشتركين

## 🛠️ التقنيات المستخدمة

### Frontend

- **React 18** - مكتبة واجهة المستخدم
- **TypeScript** - للأمان والوضوح في الكود
- **Vite** - أداة البناء السريعة
- **TailwindCSS** - للتصميم المتجاوب
- **React Router** - للتنقل بين الصفحات
- **Radix UI** - مكونات واجهة المستخدم
- **Lucide React** - مكتبة الأيقونات

### Backend

- **Supabase** - قاعدة بيانات PostgreSQL
- **Real-time subscriptions** - للتحديثات المباشرة

### UI/UX

- **Responsive Design** - يعمل على جميع الأحجام
- **Dark Mode Support** - وضع ليلي
- **RTL Support** - دعم اللغة العربية
- **Modern Design** - تصميم عصري وأنيق

## 📦 التثبيت والتشغيل

### المتطلبات الأساسية

```bash
Node.js >= 18.0.0
npm أو yarn أو pnpm
```

### خطوات التثبيت

1. **استنساخ المشروع**

```bash
git clone [رابط المستودع]
cd fusion-starter
```

2. **تثبيت التبعيات**

```bash
npm install
```

3. **إعداد متغيرات البيئة**

```bash
# إنشاء ملف .env
cp .env.example .env

# تحديث المتغيرات
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

4. **إعداد قاعدة البيانات**

```bash
# تشغيل سكريبت إعداد الجداول في Supabase
# انسخ محتويات fix-foreign-keys.sql في SQL Editor
```

5. **تشغيل التطبيق**

```bash
npm run dev
```

## 🗄️ قاعدة البيانات

### الجداول الرئيسية

- `subscribers` - بيانات المشتركين
- `course_points` - نقاط التمارين
- `diet_items` - العناصر الغذائية
- `products` - المنتجات
- `sales` - المبيعات
- `groups` - مجموعات البرامج
- `group_course_items` - ربط المجموعات بالتمارين
- `group_diet_items` - ربط المجموعات بالأنظمة الغذائية
